## 📚 What I'm Learning

This repository documents my hands-on journey learning **PyTorch** — from understanding tensors all the way to building and optimizing deep learning models for real datasets.

---

## 📁 Notebooks

| Notebook | Description |
|----------|-------------|
| [`Fundamental.ipynb`](Fundamental.ipynb) | PyTorch fundamentals — tensors, operations, autograd, and core concepts |
| [`NN_implement.ipynb`](NN_implement.ipynb) | Building a Neural Network from scratch using PyTorch |
| [`ANN_building.ipynb`](ANN_building.ipynb) | Designing and training Artificial Neural Network (ANN) architectures |
| [`CNN_architect.ipynb`](CNN_architect.ipynb) | Convolutional Neural Network (CNN) architecture design and implementation |
| [`Transfer_learning_on_Mnist_Fashion.ipynb`](Transfer_learning_on_Mnist_Fashion.ipynb) | Applying Transfer Learning on the Fashion MNIST dataset |
| [`ann_fashion_mnist_pytorch_gpu_optimized_optuna.ipynb`](ann_fashion_mnist_pytorch_gpu_optimized_optuna.ipynb) | GPU-optimized ANN on Fashion MNIST with hyperparameter tuning using Optuna |

---

## 🧠 Topics Covered

- **Tensors** — creating, reshaping, indexing, and operations
- **Autograd** — automatic differentiation and gradient computation
- **ANN Architecture** — layers, activations, loss functions, optimizers
- **CNN Architecture** — convolution, pooling, feature maps
- **Training from Scratch** — full training loops built manually
- **Transfer Learning** — leveraging pretrained models on new tasks
- **GPU Optimization** — running models on CUDA for faster training
- **Hyperparameter Tuning** — using Optuna for automated optimization
